# Report a Bug
## Overview
A bug report has come in from a customer and your task is to create a github issue for tracking the bug. If a bug report is already created for the request, add a "+1" comment on the existing issue to indicate that it's being reported again.

## Process
1. Gain an understanding of the bug being reported
2. Use the Github MCP server to find any existing issues for this bug
3. If one exists, add a "+1" comment on it, if not, create a new bug report issue using the MCP server
